# 行为准则

## 我们的承诺

为营造开放、友好的环境，我们承诺：无论年龄、体型、残障、民族、性别认同与表达、经验水平、教育背景、社会地位、外貌、种族、宗教或性取向，参与本项目的每个人都享有不受骚扰的体验。

## 我们的标准

有助于营造积极环境的行为包括但不限于：

- 使用友好、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 关注对社区最有利的方向
- 对其他社区成员保持同理心

不可接受的行为包括但不限于：

- 使用带有性暗示的言语或图像，以及不受欢迎的性关注或搭讪
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息（如住址、邮箱等）
- 其他在专业环境中可被合理认为不当的行为

## 责任

维护者有权利和义务删除、编辑或拒绝与行为准则不符的评论、提交、代码、Wiki 编辑、Issue 及其他贡献，并可暂时或永久禁止其认为行为不当、威胁、冒犯或有害的参与者。

## 适用范围

本行为准则适用于项目空间以及在与项目相关的公开场合中代表项目时的个人行为。

## 执行

可向项目维护者报告滥用、骚扰或其他不可接受的行为，维护者将酌情处理并保密。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.0。
