# 获取帮助

## 常见问题

- **安装失败**：确保已安装 Python 3.9+，并使用 `./install.sh` 创建虚拟环境。
- **Telegram 推送失败**：检查 `.env` 中 `TELEGRAM_BOT_TOKEN` 与 `TELEGRAM_CHAT_ID` 是否正确，且 Bot 已向该会话发过至少一条消息。
- **OpenClaw 报错**：确认已配置 `OPENCLAW_GATEWAY_TOKEN` 且网关可用。
